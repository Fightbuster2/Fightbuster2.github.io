    Projecten.

    Pong game

    Pong de 2D game die ik als eerste maakte in het eerste jaar als software developmenr op het Sintlucas in eindhoven. Ik heb de coding talen C# en het programma Unity gebruikt, de pong game is een game waar dat een balletje rond stuitert en je met een klein balkje omhoog en omlaag kan gaan om het tegen te houden. het is een 1vs1 game dus je kan het ook met iemand anders spelen. ik heb een extra game mode gemaakt met een extra balkje in het midden die automatisch omhoog en omlaag gaat. Je krijgt een punt als het balletje de andere kant raakt achter het balkje die je zelf omhoog en omlaag kan doen Ik heb als enige aan dit project gewerkt met een beetje hulp van de docent die mij les gaf dus ik heb het design zelf en de code zelf gemaakt. ik moest deze game maken voor een school project. ik heb er heel veel van geleerd en vond het een hele leuke tijd om aan te werken, maar ben er wel achter gekomen dat ik het toch leuker vind om websites en apps te developen

    Threadwear

    Threadwear, is de eerste website/webshop die ik gemaakt heb in het eerste jaar van deze opleiding, ik heb hier heel veel tijd in gestoken omdat ik het zelf een heel leuk en leerzaam project vond. Ik ben heel blij met het eindresultaat, de coding talen die ik heb gebruikt zijn HTML en CSS ik vind het jammer dat ik geen JavaScript heb toegevoegd maar dit kwam omdat ik daar in het eerste jaar nog geen verstand van had. Deze website/webshop is gemaakt voor mensen die schoenen willen kopen en schoenen die mensen willen verkopen, dit project heb ik moeten maken voor een school project maar was eigenlijk van plan om de website zelf te gebruiken maar daar ben ik helaas niet aan toe gekomen. de leraren waaren erg onder de indruk dat ik met de informatie en verstand van HTML en CSS toch een mooie functionele website/webshop heb gemaakt, door dit project ben ik eigenlijk overgehaald om toch richting de web en app development richting te gaan en me daar meer in te gaan verdiepen.

    Webshop

    Webshop, deze webshop is voor mij de eerste keer dat ik gebruik heb gemaakt van JavaScript. Dit project was leuk om te maken maar ik kwam er achter dat ik toch nog meer moest doen om het helemaal perfect maar uiteindelijk heb ik er het beste van gemaakt en was ik toch blij met het eindresultaat. Ik heb veel geleerd tijdens deze opdracht vooral over JavaScript, dir project was uiteindelijk meer gebaseerd op JavaScript dan hoe het eruit moest zien dus ik heb de focus niet op CSS gelegd maar vooral op JavaScript. In deze webshop kun je op de afbeelding klikken om meer informatie te zien over het product zo kon je daar ook op een knop klikken (add to cart) zodat het product daadwerkelijk in de cart komt, daarbij heb ik ook een tooltip gemaakt op de checkout button zodat de mensen weten wat er gebeurt als je daar op klikt, als je op add to cart klikt komt er een nummertje bij de cart bij en de prijs gaat omhoog, er is een max van 6 producten dan is alles out of stock. Als je checkout klikt reset de cart en heb je eigenlijk betaald voor de producten.
